## Summary

## Key Format

All keys use **dot notation** with lowercase and underscores:
```
section.subsection.key_name
```

Examples:
- `nav.home`
- `dashboard.daily_checkin`
- `auth.forgot_password`
- `common.save`

## Usage

```typescript
import { useTranslation } from "@/lib/i18n-provider";

const { t } = useTranslation();
t("nav.home")  // Returns appropriate translation
```

## Adding New Keys

1. Add to `locales/en.json`
2. Add to `locales/fr.json` (matching structure)
3. Use with `t("your.new.key")`
